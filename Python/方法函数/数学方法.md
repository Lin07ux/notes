### 1. divmod 获取除法的商和余数

使用`divmod(num1, num2)`可以计算 num1 除以 num2 后得到的商和余数。

```shell
>>> divmod(5, 2)  #表示5除以2，返回了商和余数
(2, 1)
>>> divmod(9, 2)
(4, 1)
>>> divmod(5.0, 2)
(2.0, 1.0)
```

### 2. round 四舍五入

`round(num, length)`可以使用四舍五入法则保留参数 num 的指定位数的小数。

```shell
>>> round(1.234567, 2)
1.23
>>> round(1.234567, 3)
1.235
>>> round(10.0/3, 4)
3.3333
>>> round(1.234)
1.0
>>> round(1.2345, 3)
1.234 # 应该是：1.235
>>> round(2.235, 2)
2.23  # 应该是：2.24
```

可以看到，最后的两个四舍五入并不符合预期，这是由于二进制与十进制之间的转换造成的。

### 3. abs 绝对值

`abs()`获取数值的绝对值。

```shell
>>> abs(10)
10
>>> abs(-10)
10
>>> abs(-1.2)
1.2
```
